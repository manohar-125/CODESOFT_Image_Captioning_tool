# 🖼️ Task 3 – Image Captioning using Transformers

This project is part of the **CodSoft Artificial Intelligence Internship**.

## 📌 Objective
Use a pre-trained Vision + Language model to generate text captions for images.

## ⚙️ Tech Stack
- Python
- Hugging Face Transformers (`vit-gpt2-image-captioning`)
- PyTorch
- Google Colab

## 🧠 Features
- Upload any image
- Automatically generates relevant English captions
- Uses Vision Transformer + GPT2 architecture (via Hugging Face)

## 🚀 How to Run (Colab Recommended)
1. Open the notebook in Google Colab
2. Upload an image when prompted
3. Run all cells to see caption output

📂 File: `task3_image_captioning.ipynb`
